# Discord Topluluğunuz için harika botlar kurun
## Önemli notlar:
* altyapı ubuntu vdslerde çalışmak için tasarlandı.
* altyapıyı çalıştırmal için pm2'yi global olarak kurmalısınız.
```bash
npm install pm2 -g
```
* node.js 16 veya daha yüksek olmalı.
```bash
node -v
```
